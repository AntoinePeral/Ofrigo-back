LISTE DES ROUTES DISPONIBLE :

===Account===

- Public :

/register POST Crée un compte
/login POST Se connecter

- User :

/me/profile GET Données utilisateur connecté
/me/profile PUT Modifie les données d'un utilisateur
/me/profile DELETE Supprime un utilisateur
/me/profile/ingredient POST Ajoute un ingredient a l'account
/me/profile/ingredient/:id DELETE Supprime un ingredient de l'account

- Admin :

/admin/profile GET Affiche tous les comptes utilisateur
/admin/profile/:id GET Affiche un compte utilisateur
/admin/profile/:id DELETE Supprime un compte utilisateur

===Category===

- Public :

/category GET Affiche toutes les categories
/category/:id GET Affiche une categories

- Admin :

/admin/category GET Affiche toutes les categories
/admin/category/:id GET Affiche une categories
/admin/category POST Crée une categorie
/admin/category/:id PUT Modifie une categorie
/admin/category/:id DELETE Supprime une categorie

===Ingredient===

- Public :

/ingredient GET Affiche tous les ingrédients
/ingredient/:id GET Affiche un ingrédient

- User :

/me/profile/ingredient GET Affiche tous les ingredients d'un utilisateur
/me/profile/:id GET Affiche un ingredient d'un utilisateur

- Admin :

/admin/ingredient GET Affiche tous les ingrédients
/admin/ingredient/:id GET Affiche un ingrédient
/admin/ingredient POST Crée un ingrédient
/admin/ingredient PUT Modifie un ingrédient
/admin/ingredient DELETE Supprime un ingrédient

===Message===

- Public :

/contact POST Envoyer un message

- User :

/me/profile/message GET Affiche toutes les messages d'un utilisateur
/me/profile/message/:id GET Affiche un message d'un utilisateur
/me/profile/contact POST Envoyer un message en tant qu'utilisateur
/me/profile/message/:id PUT Modifie un message de l'utilisateur
/me/profile/message/:id DELETE Supprime un message de l'utilisateur

- Admin :

/admin/message GET Affiche tous les messages
/admin/message/:id GET Affiche un message
/admin/contact POST Envoyer un message en tant qu'admin
/admin/message/:id PUT Modifie un message de l'admin
/admin/message/:id DELETE Supprime un message de l'admin

===Recipe===

- Public :

/recipe GET Affiche toutes les recettes
/recipe/:id GET Affiche une recette

- Admin :

/admin/recipe GET Affiche toutes les recettes
/admin/recipe/:id GET Affiche une recette
/admin/recipe POST Créer une recette (seul)
/admin/recipe/:id PUT Modifie une recette
/admin/recipe/:id DELETE Supprime une recette

===Tag==

- Public :

/tag GET Affiche toutes les tags
/tag/:id GET Affiche un tag

- Admin :

/admin/tag GET Affiche toutes les tags
/admin/tag/:id GET Affiche un tag

/admin/tag/:id PUT Modifie un tag
/admin/tag/:id DELETE Supprime un tag
